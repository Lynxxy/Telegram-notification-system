# Telegram Notification System

Система уведомлений в Telegram. Бот отправляет сообщения и медиа файлы (фото, аудио, видео и т.д.).

## Установка

1. Клонируйте репозиторий.
2. Установите зависимости:
pip install -r requirements.txt
3. Создайте файл `.env` и добавьте ваш токен бота и chat_id:
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_CHAT_ID=your_chat_id_here

## Запуск бота

Для запуска бота используйте Python 3.7 или выше, так как проект использует асинхронный код.

1. Убедитесь, что у вас установлен Python 3.7+.
2. Запустите бота:
python bot.py

Бот будет отправлять приветственное сообщение в указанный чат и будет готов к обработке команд.

## Функции

- Отправка текстовых сообщений
- Отправка медиа (фото, видео, аудио)
- Отправка местоположения
- Отправка контактов
- Обработка команд (например, `/start`)

## Зависимости

- `python-telegram-bot` (с поддержкой асинхронности)
- `python-dotenv` для загрузки переменных окружения

## Примечания

Бот использует асинхронную обработку с использованием библиотеки `python-telegram-bot` и `asyncio`. Это позволяет эффективно обрабатывать запросы и отправлять сообщения в реальном времени.
